## Fiches de référence rapide

Ce dossier regroupe des aides‑mémoire (cheat‑sheets) pour accélérer la prise en main de n8n.

### 1. Fiche « master list » des nœuds

Une feuille Google Sheets créée par la communauté recense plus de mille nœuds n8n avec leur type (core, trigger, régulier), une description succincte et un lien vers la documentation officielle.  
C’est un excellent point de départ pour savoir si un nœud existe déjà pour votre service et pour découvrir de nouveaux connecteurs.

### 2. Cheat‑sheet « data pipeline »

Cette fiche, orientée data science, met en avant les nœuds indispensables pour construire des pipelines d’acquisition et de transformation de données :  
• automatisation de l’ingestion de fichiers (FTP, e‑mail, API),  
• transformation via les nœuds de code (JavaScript/Python),  
• agrégation et export (base de données, Google Sheets, CSV),  
• notifications sur les erreurs.  
C’est un outil pratique pour les équipes qui migrent des scripts traditionnels vers une approche low‑code.

### 3. Cheatsheet générale (PDF)

Un membre de la communauté a compilé dans un PDF haute résolution les éléments essentiels de n8n :  
• déclencheurs les plus utilisés (Webhook, Cron, Email, etc.),  
• expressions et variables de workflow,  
• paramètres courants des nœuds (timeout, retry, error handling),  
• raccourcis clavier de l’éditeur,  
• astuces de débogage et d’optimisation.  
Le fichier est téléchargeable depuis la communauté. Veillez à respecter les conditions d’utilisation et de partage.

> **Astuce** : ajoutez vos propres notes ici ou créez de nouvelles fiches à mesure que vous découvrez des fonctions utiles.
